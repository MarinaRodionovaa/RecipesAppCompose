Это приложение — кулинарный справочник с удобной навигацией по категориям. 
Пользователь выбирает тип блюда, просматривает подборку рецептов с фото, открывает подробную карточку с ингредиентами, регулировкой порций и пошаговым приготовлением. 
Есть раздел «Избранное» для быстрого доступа к любимым рецептам.
